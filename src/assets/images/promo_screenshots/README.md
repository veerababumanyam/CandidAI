# Promotional Screenshots for Chrome Web Store

This directory contains promotional screenshots and images for the Chrome Web Store listing.

## Required Screenshots

The Chrome Web Store requires the following screenshots:

1. **Main Screenshot** (1280x800 or 640x400)
   - Shows the main interface of CandidAI during an interview
   - Displays the side panel with transcription and suggestions
   - Filename: `main_screenshot.png`

2. **Features Screenshot** (1280x800 or 640x400)
   - Highlights key features like visual analysis and calendar
   - Shows multiple features in action
   - Filename: `features_screenshot.png`

3. **Settings Screenshot** (1280x800 or 640x400)
   - Shows the settings page with configuration options
   - Highlights customization capabilities
   - Filename: `settings_screenshot.png`

4. **Performance Hub Screenshot** (1280x800 or 640x400)
   - Shows the performance hub with analytics
   - Displays interview history and insights
   - Filename: `performance_screenshot.png`

5. **Calendar Screenshot** (1280x800 or 640x400)
   - Shows the calendar interface with interview events
   - Highlights scheduling capabilities
   - Filename: `calendar_screenshot.png`

## Promotional Images

The Chrome Web Store also requires promotional images:

1. **Small Promotional Tile** (440x280)
   - Used in the Chrome Web Store search results
   - Simple, clear design with logo and tagline
   - Filename: `small_promo.png`

2. **Large Promotional Tile** (920x680)
   - Used on the Chrome Web Store detail page
   - More detailed design showcasing key features
   - Filename: `large_promo.png`

3. **Marquee Promotional Tile** (1400x560)
   - Used for featured extensions
   - Wide format showcasing the extension in use
   - Filename: `marquee_promo.png`

## Guidelines for Screenshots

When creating screenshots:

1. **Content**:
   - Show the extension in action with realistic content
   - Highlight key features and benefits
   - Ensure text is readable

2. **Quality**:
   - Use high-resolution images (2x for retina displays)
   - Ensure proper lighting and contrast
   - Avoid compression artifacts

3. **Branding**:
   - Use consistent colors from the CandidAI brand palette
   - Include the CandidAI logo where appropriate
   - Maintain consistent typography

4. **Privacy**:
   - Do not include real user data or personally identifiable information
   - Use placeholder or mock data for demonstrations
   - Blur or remove any sensitive information

## How to Create Screenshots

1. Install the extension in Chrome
2. Set up the extension with mock data
3. Navigate to the relevant screens
4. Use Chrome's built-in screenshot tool or a third-party tool
5. Edit the screenshots to highlight key features
6. Save in the required dimensions and formats

## Chrome Web Store Requirements

The Chrome Web Store has specific requirements for promotional materials:

- All images must be in PNG or JPEG format
- No text in the small promotional tile
- Limited text in the large promotional tile
- No promotional borders, badges, or calls-to-action
- No mimicking Chrome UI elements
- No screenshots of other Google or third-party properties without permission

For more details, see the [Chrome Web Store Developer Documentation](https://developer.chrome.com/docs/webstore/images/).
